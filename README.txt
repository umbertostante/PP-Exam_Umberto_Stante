Questo progetto implementa una classe Python chiamata MovieLibrary, progettata per 
gestire una collezione di film memorizzati in un file JSON, che offre funzionalità 
per aggiungere, rimuovere, modificare e filtrare film, e garantisce che tutte le 
operazioni siano immediatamente riflesse nel file JSON associato.

FILE PRINCIPALI:

1. movie_library.py :	Contiene il codice con le funzionalità richieste implementate.
2. movies.json : 	File JSON contenente la collezione di film, usato per leggere e
		  	salvare le modifiche.

Tutte le prove effettuate durante le scrittura del codice sono state eliminate, per
fornire, in fase di consegna, un codice pulito.
